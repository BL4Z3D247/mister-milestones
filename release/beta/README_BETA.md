# MiSTer Milestones Beta

## Install
chmod +x install.sh
./install.sh

## Verify
./check.sh

## Run
./run.sh

## Uninstall
./uninstall.sh

---

## What This Does

• Reads MiSTer core RAM via /dev/mmr_memtap  
• Runs RetroAchievements runtime  
• Logs achievements  
• Does NOT modify system  
• Safe to uninstall  

---

## Safety

This beta only reads memory.
It does not write to MiSTer.
It does not modify cores.
It does not patch FPGA bitstreams.
