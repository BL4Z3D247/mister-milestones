# MiSTer Milestones v0.1.0-a2 — Hardware Alpha

**Tag:** v0.1.0-a2  
**Scope:** daemon hardening + shared ABI alignment + strict loopback device

## What’s in this release
- Hardened daemon runtime behavior and logging
- Aligned userspace/kernel memtap ABI usage
- Strict loopback `/dev/mmr_memtap` implementation backed by snapshot files
- Improved argument handling and validation paths

## Intended use
This is a **hardware alpha validation** package. It enables:
- Mock-mode validation of the evaluation pipeline
- Loopback device-mode validation without FPGA core patches
